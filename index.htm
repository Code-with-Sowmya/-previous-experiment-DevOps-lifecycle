DevOps Lifecycle for Word Wander

Plan

Define game rules, features, and UI ideas. Decide tech stack (HTML, CSS, JS).

Develop

Write game logic and UI code. Collaborate on GitHub with your teammate.

Build

Organize code, optimize files, and prepare structure for deployment.

Test

Check word guessing logic, error handling, and fix bugs in gameplay.

Release

Push the final version to GitHub with tags and final review.

Deploy

Use Vercel to host the app directly from the GitHub repository.

Operate

Ensure app runs smoothly, is available, and loads properly for users.

Monitor

Use GitHub Issues and feedback to track bugs and improvements.

➡️ Plan → Develop → Build → Test → Release → Deploy → Operate → Monitor ➡️
